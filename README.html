簿記 商店経営RPG v2 — 導入手順

  1. このフォルダをGitHubにアップ → Settings → Pages → Branch: main / Folder: root → Save
  2. 公開URLが出たら開く
  3. ホーム画面の入力欄に、Googleスプレッドシートの 公開CSV URL を貼って「保存」

Googleスプレッドシート側の設定

  1. 問題を1枚のシートに作成（列は下記「列仕様」を参照）
  2. ファイル → ウェブに公開 → 対象シートを選択 → 形式「CSV」 → 公開
  3. 表示されたURL（CSV）をアプリの入力欄に貼り「保存」

列仕様（五分類＋その他／四択のみ）

id / ユニット / タグ / 問題タイプ / 問題文 / 選択肢A / 選択肢B / 選択肢C / 選択肢D / 正解 / 解説 / 例解仕訳_借方科目 / 例解仕訳_借方金額 / 例解仕訳_貸方科目 / 例解仕訳_貸方金額

分類問題の選択肢は「資産 / 負債 / 純資産 / その他」を推奨（収益・費用など五分類外は「その他」に内包）。

オフライン・保存

  • PWA対応：初回アクセス後はオフラインでも起動可能
  • 進捗（週ポイント、レベル、今日のXP）はローカル保存（localStorage）
  • 後日、Googleログイン＋クラウド同期に拡張可能
